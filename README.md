# psychoApps 👶

子育てフレーズ集 - Streamlitベースの子育て支援アプリケーション

## 概要

psychoAppsは、子育てでよく使うフレーズを簡単に検索・コピーできるWebアプリケーションです。
Streamlitを使用して構築されており、保育園への連絡、祖父母への依頼、パートナーとの共有など、
様々な場面で使える子育てフレーズを100以上収録しています。

## 機能

- 👶 100以上の子育てフレーズを11カテゴリに分類
- 🔍 キーワード検索で素早くフレーズを発見
- 📂 カテゴリフィルターで絞り込み表示
- 📋 ワンクリックでフレーズを選択・コピー
- 📱 レスポンシブデザインでスマホにも対応
- 💬 LINEなどのメッセージアプリにすぐ送信可能

## セットアップ

### 前提条件

- Python 3.8以上

### インストール

1. リポジトリをクローン
```bash
git clone <repository-url>
cd psychoApps
```

2. 仮想環境を作成・有効化
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

3. 依存関係をインストール
```bash
pip install -r requirements.txt
```

### 実行

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` にアクセスしてください。

## フレーズカテゴリ

以下の11カテゴリ、100以上のフレーズを収録しています：

1. **朝の挨拶・起床** - 朝の声かけやモーニングルーティン
2. **食事** - 食事中の声かけやマナー指導
3. **遊び・外出** - 公園や外出時のコミュニケーション
4. **励まし・ほめる** - ポジティブなフィードバック
5. **しつけ・注意** - やさしく伝える注意の言葉
6. **お風呂** - バスタイムの声かけ
7. **就寝・夜** - 寝る前のルーティンとおやすみの挨拶
8. **トイレ・おむつ** - トイレトレーニングサポート
9. **お出かけ準備** - 外出準備の促し
10. **感情のサポート** - 子どもの気持ちに寄り添う言葉
11. **お手伝い・生活習慣** - 自立を促す声かけ

## 使い方

1. アプリを起動後、左のサイドバーから「子育てフレーズ集」を選択
2. **検索**: キーワードで検索ボックスから絞り込み（例: 「おはよう」「食事」）
3. **カテゴリ選択**: サイドバーから特定のカテゴリを選択
4. **フレーズ選択**: 使いたいフレーズをクリック
5. **コピー**: 表示されたフレーズをコピーしてLINEなどに送信
